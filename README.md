This is just a little example about using Expressjs with backbone + Marionette

In the projecto you will to find examples about the use of:
 - Backbonejs
 - Marionettejs
 - Expressjs
 - Mongoose
 - Bootstrap
 - Requirejs
 - Mocha testing

1.- Database
==================
    Install mongodb and start daemon

2.- Backend
==================

## setup

    npm install


## Run tests

    npm test

## Start app

    node app.js



2.- Frontend
==================
## setup

    npm install
    bower install


## Run tests

    npm test
